# desenvolvimento-web-Grupo-E

#Resumo - Aula 01
Disciplina: Desenvolvimento Web

---

## 1. Aprentação da Disciplina
A disciplina de Desenvolvimento Web combina princípios de design visual com tecnologia web para criar interfaces digitais funcionais e atrativas.

---

## 2. Objetivos da Disciplina 
- Compreender conceitos fundamentais do desenvolvimento web
- Retomar HTML, CSS, JavaScript
- Entender versionamento e deploy
- Desenvolver um projeto completo ao longo do semestre

---

## 3. Controle de Versão 
Controle de versão é o registro das alterações realizadas em arquivos ao longo do tempo. 

### Benefícios: 
- Recuperar verções específicas
- Trabalhar em equipe
- Criar ramificações (branches)

### Principais plataformas:
- GitHub
- GitLab
- Bitbucket

---

## 4. Versionamento e Deploy 
Deploy é o proceso de publicar uma aplicação em um servidor para qe fique acessivel online.

Também foram abordados:
- CI/CD (Integração Contínua)
- Automação de deploy
- Monitoramento de aplicções

---

## 5. Conteúdos do Semestre
- HTML
- CSS
- JavaScript
- Frameworks Front-end (React, Vue, Angular)
- Frameworks Back-end
- Banco de dados no SQL
- UX e Prototipação

  ## 6. Avaliação
  - AC1 - 15% (Prova Teórica)
  - AC2 - 30% (Aprensentação sobre o projeto)
  - AG - 10%
  - AF - 45% (Documentação completa do projeto/Mini TCC)

---

## 7. Importância do Portifólio
Ter um portifólio demostra habilidades práticas, criatividade e domínio das ferramentas utilizadas no desenvolvimento web.
